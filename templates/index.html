Fantasy Basketball Tools

An application that calculates and displays various statistics for ESPN Fantasy Basketball leagues. The code/application works for (public) leagues of any size that use H2H Most Cat. scoring and a scoring system of FG%, FT%, 3PM, REB, AST, STL, BLK, TO, PTS. For more information and to see the source code of the project, checkout the GitHub repository.
